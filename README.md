## สรุปสั้น ๆ ว่ามันคืออะไร

1. สมัคร 1 บัญชี → สลับได้ 2 บทบาท (จ้างคน / รับงาน)
2. **ผู้ว่าจ้าง**: โพสงานดูแลผู้สูงอายุ + ปักหมุด GPS + ตั้งงบ
3. **แคร์กิฟเวอร์**: ยืนยันตัวตน (KYC) → แอดมินอนุมัติ → เห็นงานบนแผนที่ → กดขอรับงาน
4. หลายคนกดขอได้ → ผู้ว่าจ้างดูดาว/โปรไฟล์ แล้วเลือก 1 คน
5. คุยตกลงเวลา/ราคาในแชท → ทำงาน → **จ่ายเงินกันเอง (ไม่มีเงินวิ่งผ่านระบบ)**
6. ปิดงาน → ให้ดาวกัน

### ⭐ กฎความปลอดภัยที่เป็นหัวใจของแอพ
