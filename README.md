## สรุปสั้น ๆ ว่ามันคืออะไร

1. สมัคร 1 บัญชี → สลับได้ 2 บทบาท (จ้างคน / รับงาน)
2. **จ้างคนได้ 2 ทาง:**
   - **จ้างตรง** — เดินดูโปรไฟล์แคร์กิฟเวอร์ → ยิงคำขอไปหาคนที่ถูกใจ → เขากดรับ/ปฏิเสธ
   - **โพสงาน** — โพส + ปักหมุด GPS + ตั้งงบ → หลายคนกดขอรับ → ผู้ว่าจ้างเลือกเอง 1 คน
3. **แคร์กิฟเวอร์** ต้องยืนยันตัวตนก่อน ถึงจะรับงานได้ + เห็นที่อยู่จริง + โผล่ในไดเรกทอรี
4. คุยตกลงเวลา/ราคาใน **แชทสด** (ส่งรูปได้ · รู้ว่าอ่านยัง · รู้ว่าใครออนไลน์) → ทำงาน → **จ่ายเงินกันเอง (ไม่มีเงินวิ่งผ่านระบบ)**
5. ปิดงาน

### ⭐ กฎความปลอดภัยที่เป็นหัวใจของแอพ

**แคร์กิฟเวอร์ที่ยังไม่ยืนยันตัวตน จะไม่เห็นที่อยู่บ้านผู้สูงอายุ** — เห็นแค่วงกลมกว้าง 800 ม.

เพราะที่อยู่บ้านที่มีคนแก่อยู่ลำพังเป็นข้อมูลอ่อนไหว ไม่ควรเปิดให้ใครก็ได้ที่โหลดเว็บเห็น

## ⚠️ เรื่องที่ต้องรู้ก่อนแตะโค้ด

| เรื่อง | รายละเอียด |
|---|---|
| **KYC เป็นโหมดเดโม** | กดปุ่มเดียวผ่านทันที ไม่มีอัปบัตร ไม่มีแอดมินอนุมัติ — แต่ **กฎ GPS 2 ระดับยังทำงานเต็มร้อย** |
| **รีวิว/ให้ดาวปิดอยู่** | คอมเมนต์ไว้ที่ `server.js` — โค้ดกับตารางยังอยู่ครบ ⭐ เลยขึ้น 0.0 ทุกที่ |
| **Flutter ยกเลิกแล้ว** | ลบโฟลเดอร์ทิ้งแล้ว (2026-07-17) |
| 🆕 **แชทใช้ Socket.IO** | ทะเบียน "ใครออนไลน์" อยู่ใน RAM → ใช้ได้กับ **pm2 fork โปรเซสเดียว** เท่านั้น<br>เปลี่ยนเป็น cluster เมื่อไหร่ จุดเขียวเพี้ยนทันที |
| 🆕 **`SET time_zone` ใน `src/db.js`** | **อย่าลบ** — เคยทำเวลาทุกที่ในเว็บล้ำหน้าไป 7 ชม. (แก้ 2026-07-17) |
| **`.env` + `uploads/` ไม่อยู่บน git** | มีที่เดียวบน server — **ลบโฟลเดอร์ทิ้งโดยไม่สำรอง = แอพต่อ DB ไม่ได้ + รูปในแชทหายหมด** |
| 🔴 **repo เป็น public** | เอกสารในนี้มี IP + ผังเครื่อง — พี่ดิวรับทราบและเลือกเอง |
| 🔴 **รหัส SSH หลุดในแชท AI** | ยังไม่ได้เปลี่ยน — ดู [01_SERVER_INFO.md](01_SERVER_INFO.md) |
