## 🎬 สคริปต์เดโม (10 นาที เห็นครบทุกฟีเจอร์)

### ฉากที่ 1 — จ้างตรง (โชว์ก่อน เพราะเห็นผลเร็วสุด)

1. ล็อกอิน `emp@test.com` → เลือก **"ต้องการจ้างแคร์กิฟเวอร์"**
2. แท็บ **"เลือกคนเอง"** → **เห็นแคร์กิฟเวอร์ 5 คนพร้อมเรต ย่าน ทักษะ ประวัติ**
3. ลองพิมพ์ค้นหา *"ติดเตียง"* → เหลือแต่คนที่ถนัดผู้ป่วยติดเตียง
4. เลือก **บุญมี ใจเย็น** (ประสบการณ์ 8 ปี) → กด **"ส่งคำขอจ้าง"**
5. กรอกหัวข้อ + งบ (ระบบเติมเรตของเขามาให้แล้ว) → **ส่งคำขอจ้าง**

> 💬 พูดตอนโชว์: *"ไม่ต้องโพสประกาศ ไม่ต้องรอ เห็นคนที่ถูกใจก็ยิงไปหาเลย"*

### ฉากที่ 2 — ฝั่งแคร์กิฟเวอร์ได้รับคำขอ

6. เบราว์เซอร์อีกตัว → ล็อกอิน `care2@demo.com` → เลือก **"เป็นแคร์กิฟเวอร์"**
7. แท็บ **"ภารกิจของฉัน"** → เห็นการ์ด**ขอบเหลือง** + ป้าย **"รอตอบรับ"** + ตัวเลขแดงบนแท็บ
8. กด **"ตอบรับงาน"** → ป้ายเปลี่ยนเป็น **"จับคู่แล้ว"**

### ฉากที่ 3 — ⭐ กฎความปลอดภัย (จุดขายที่แท้จริงของแอพ)

9. เบราว์เซอร์ที่ 3 (หรือโหมดไม่ระบุตัวตน) → **สมัครบัญชีใหม่**
10. เลือก **"เป็นแคร์กิฟเวอร์"** → แท็บ **"หางาน"**
11. **ชี้ให้ดูชัด ๆ:**
    - แถบเหลืองเตือน *"ยังไม่ได้ยืนยันตัวตน"*
    - บนแผนที่เห็นแค่ **วงกลมสีฟ้ากว้าง 800 ม.** → **ไม่รู้ว่าบ้านอยู่ตรงไหน**
    - ในรายการงานเขียน *"🔒 ตำแหน่งโดยประมาณ"* — **ไม่มีที่อยู่**
12. กด **"ขอรับงาน"** → ❌ ขึ้นเตือนว่าต้องยืนยันตัวตนก่อน

> 💬 **นี่คือประโยคที่ต้องพูด:**
> *"คนแปลกหน้าที่ยังไม่ผ่านการตรวจสอบ จะไม่มีทางรู้เลยว่าบ้านไหนมีคนแก่อยู่ลำพัง
> เห็นแค่วงกลมกว้าง 800 เมตร — เพราะที่อยู่บ้านคนแก่ ไม่ใช่ข้อมูลที่ใครก็ควรเห็น"*

### ฉากที่ 4 — ยืนยันตัวตนแล้วโลกเปลี่ยน

13. แท็บ **"บัตรแคร์กิฟเวอร์"** → กรอกประสบการณ์ / ย่าน / เรต / ทักษะ / แนะนำตัว
14. กด **"ยืนยันตัวตน"** → ✅ ผ่านทันที
15. กลับแท็บ **"หางาน"** → กด **"ค้นหางาน"** → **เทียบกับฉากที่ 3:**
    - วงกลม → กลายเป็น **หมุดปักตรงบ้านจริง** 📍 พร้อมป้ายราคา
    - **เห็นที่อยู่เต็มแล้ว**
    - แถบเหลืองหายไป
16. กดหมุด → เด้งแผ่นรายละเอียด → **"ขอรับงานนี้"** → ✅ สำเร็จ

> ⚠️ **โหมดเดโม** — ระบบจริงต้องถ่ายบัตร ปชช. + เซลฟี่ แล้วรอแอดมินตรวจ
> ตอนนี้ปิดไว้เพื่อให้เดโมง่าย **แต่กฎ GPS 2 ระดับยังทำงานเต็มร้อยเหมือนเดิมทุกประการ**

### ฉากที่ 5 — ประกาศหาผู้ดูแลแบบเปิดรับ + เลือกคนเอง

17. ฝั่ง `emp@test.com` → แท็บ **"หาผู้ดูแล"** → กรอกรายละเอียด
18. **แตะบนแผนที่เพื่อปักหมุด** ← ไม่ปักจะโพสไม่ได้
19. โพส → รอฝั่งแคร์กิฟเวอร์กด "ขอรับงาน" (ฉากที่ 4 ข้อ 16)
20. **"ภารกิจของฉัน"** → **"ดูผู้สมัคร"** → เห็นประวัติ + ประสบการณ์ทุกคน → **"เลือกคนนี้"**
    - งาน → **"จับคู่แล้ว"** · คนอื่นถูกปฏิเสธอัตโนมัติในคำสั่งเดียว

### ฉากที่ 6 — แชทต่อรอง 🆕 (ยกเครื่องใหม่ 2026-07-17)

> 💡 **ฉากนี้โชว์สวยที่สุดถ้าเปิด 2 เครื่อง/2 เบราว์เซอร์วางคู่กัน** (เช่น Chrome + Chrome ไม่ระบุตัวตน)
> ล็อกอินคนละฝั่ง แล้วให้คนดูเห็นทั้ง 2 จอพร้อมกัน — ของใหม่ทั้งหมดเป็น "สด" ต้องเห็น 2 ฝั่งถึงจะรู้สึก

21. แคร์กิฟเวอร์ → แท็บ **"แชท"** → พิมพ์ *"สนใจครับ ขอ 1,300 ได้ไหม"*
22. **จอผู้ว่าจ้าง: ข้อความเด้งขึ้นทันที** (ไม่ต้องรอ ไม่ต้องกดรีเฟรช) + ตัวเลขแดงขึ้นบนแท็บ
23. **ชี้ให้ดูจุดเขียวข้างชื่อ** = อีกฝั่งออนไลน์อยู่ · ปิดจอนั้นไป จุดจะเปลี่ยนเป็นเทาแล้วขึ้น *"เห็นล่าสุดเมื่อสักครู่"*
24. **เริ่มพิมพ์ค้างไว้ อย่าเพิ่งส่ง** → อีกจอขึ้น *"กำลังพิมพ์..."* ใต้ชื่อ
25. **ดูติ๊กใต้ฟองข้อความตัวเอง** — 🕐 กำลังส่ง → ✓ ส่งแล้ว → **✓✓ อ่านแล้ว (เปลี่ยนสีทันทีที่อีกฝั่งเปิดอ่าน)**
26. **กดปุ่มรูป 🖼 ข้างช่องพิมพ์ → ส่งรูป** (เช่น รูปสภาพห้องผู้สูงอายุ / ตารางยา)
    - รูปจากมือถือใบละ 5 MB **ถูกย่อที่เครื่องก่อนอัป เหลือหลักร้อย KB** → ขึ้นไวแม้เน็ตช้า
    - กดที่รูป = เปิดดูเต็ม ๆ ในแท็บใหม่
27. ตอบกลับ → ต่อรองกันไป-มา

> 🔒 **จุดขายเวลามีคนถามเรื่องความปลอดภัย:** รูปในแชท **ไม่ได้อยู่ใน URL ที่เดาได้**
> ต้องเป็นคู่สนทนาเท่านั้นถึงเปิดดูได้ (คนอื่นกดเข้าไป = ขึ้น "ไม่มีสิทธิ์ดูรูปนี้")
> และ **พิกัด GPS ที่ฝังมากับรูปถ่ายถูกทิ้งไปตั้งแต่ก่อนอัป** — บ้านผู้สูงอายุไม่หลุดไปกับรูป

### ฉากที่ 7 — โปรไฟล์ + ปิดงาน

28. กดไอคอน **👤** บนหัว → **หน้าโปรไฟล์** → 7 กล่อง 31 ช่อง
    - ใส่วันเกิด → **อายุคำนวณให้เอง**
    - ใส่เลขบัตร ปชช. → **พิมพ์ผิดขึ้นเตือนแดงทันที** (ตรวจหลักตรวจสอบจริง)
    - แถบ % ความสมบูรณ์ขยับตามที่กรอก
29. *(สมมติทำงานเสร็จ + จ่ายเงินสดกันเรียบร้อย — **ไม่มีเงินวิ่งผ่านแอพ**)*
30. ผู้ว่าจ้าง → **"ภารกิจของฉัน"** → **"งานเสร็จแล้ว"** → สถานะ = **"เสร็จแล้ว"**

> ⏸ **ระบบให้ดาวปิดอยู่** (ตกลงกัน 2026-07-14) — โค้ดกับตารางยังอยู่ครบ เปิดกลับได้ทันที
> คะแนน ⭐ ที่เห็นในหน้าต่าง ๆ จะเป็น 0.0 เสมอ

### ฉากที่ 8 — แอดมิน

31. ล็อกอิน `emp@test.com` → กดไอคอน **🛡** มุมขวาบน
32. เห็นรายชื่อแคร์กิฟเวอร์ทั้งหมด + สถานะ + จำนวนงานที่ขอรับไป
33. กด **"เพิกถอนการยืนยัน"** ของใครสักคน → **กลับไปดูฝั่งเขา: เห็นแค่วงกลมอีกแล้ว**

> 💬 *"แอดมินกดปุ่มเดียว คนคนนั้นกลับไปมองไม่เห็นที่อยู่บ้านลูกค้าทันที"*

## ⚠️ ก่อนโชว์จริง

1. **เตรียมงานตัวอย่างไว้ 3-5 งาน กระจายหลายย่าน** → แผนที่จะดูมีชีวิตชีวา
2. **เช็คว่าแคร์กิฟเวอร์ตัวอย่างยังอยู่ครบ**
   ```bash
   ssh server_live@192.168.1.35 "cd ~/timecarehub/TimecareHub && npm run seed"
   ```
3. **เปิดเว็บทิ้งไว้ก่อน 1 รอบ** — ให้แผนที่ OSM โหลด tile มาแคชไว้ จะได้ไม่กระตุกตอนโชว์
4. **เครื่องที่เปิดเว็บต้องต่อเน็ตได้** — Leaflet + tile แผนที่โหลดจากอินเทอร์เน็ต
